# 作品集网站使用说明

## 📁 文件结构

```
portfolio/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式文件
├── js/
│   └── app.js          # 交互功能
├── images/             # 存放图片文件
├── audio/              # 存放音频文件
└── 使用说明.md         # 本说明文件
```

## 🚀 如何打开网站

### 方法1：直接打开（最简单）
双击 `index.html` 文件，用浏览器打开

### 方法2：启动本地服务器
双击 `start.bat` 文件，然后在浏览器访问 `http://localhost:8000`

## 📸 如何添加图片作品

### 步骤1：准备图片
- 将你的图片文件放入 `images/` 文件夹
- 支持的格式：jpg, jpeg, png, gif, webp
- 建议尺寸：宽度 800px 以上，文件大小不超过 5MB

### 步骤2：添加作品
1. 在网页中点击 **"添加作品"** 按钮
2. 填写表单：
   - **作品标题**：给你的作品起个名字
   - **作品分类**：选择 "设计" 或 "摄影"
   - **作品描述**：详细介绍你的作品
   - **标签**：用逗号分隔，如：风景, 夜景, 城市
   - **图片路径**：填写 `images/你的图片文件名.jpg`
     - 例：`images/myphoto.jpg`
     - 也可以使用网络图片链接

### 示例
```
作品标题：夕阳下的海边
作品分类：摄影
作品描述：这是在三亚拍摄的海边日落照片...
标签：风景, 海边, 日落
图片路径：images/sunset.jpg
```

## 🎵 如何添加音频作品

### 步骤1：准备音频文件
- 将你的音频文件放入 `audio/` 文件夹
- 支持的格式：mp3, wav, ogg
- 建议文件大小不超过 20MB

### 步骤2：添加作品
1. 点击 **"添加作品"** 按钮
2. 填写表单：
   - **作品分类**：选择 **"音频"**
   - **音频文件路径**：填写 `audio/你的音频文件名.mp3`
     - 例：`audio/mymusic.mp3`
   - **图片路径**（可选）：可以为音频作品配一张封面图

### 示例
```
作品标题：夏日微风
作品分类：音频
作品描述：原创音乐作品，表达夏日的清凉感...
标签：音乐, 原创, 轻音乐
图片路径：images/music_cover.jpg
音频文件路径：audio/summer.mp3
```

## 💡 使用技巧

### 1. 图片优化建议
- 使用在线工具压缩图片（如 https://tinypng.com）
- 推荐尺寸：1200x800 像素
- 命名规范：使用英文，不要有空格，如 `my-work-01.jpg`

### 2. 标签使用
- 标签可以帮助分类和搜索
- 建议每个作品 3-5 个标签
- 标签示例：
  - 摄影：风景、人像、街拍、黑白、夜景
  - 设计：Logo、UI、海报、插画、品牌
  - 音频：音乐、原创、轻音乐、电子、古典

### 3. 文件命名规范
```
✅ 推荐：
- my-photo-01.jpg
- landscape_sunset.png
- summer_music.mp3

❌ 避免：
- 我的照片.jpg（中文）
- photo 1.jpg（含空格）
- IMG_20240507_143022.jpg（太长）
```

## 🔧 常见问题

### Q1: 图片显示不出来？
- 检查图片是否放在 `images/` 文件夹
- 检查文件名是否拼写正确（区分大小写）
- 检查文件格式是否正确

### Q2: 音频无法播放？
- 确认音频文件放入 `audio/` 文件夹
- 检查音频格式是否为 mp3/wav/ogg
- 检查浏览器是否支持该音频格式

### Q3: 如何修改已添加的作品？
目前需要手动编辑 `js/app.js` 文件中的 `worksData` 数组

### Q4: 可以添加视频吗？
目前支持的视频功能较简单，建议：
- 上传视频到视频网站（如B站、YouTube）
- 在描述中添加视频链接

## 🎨 自定义配色

如需修改网站颜色，编辑 `css/style.css` 文件中的 CSS 变量：

```css
:root {
    --primary-color: #667eea;      /* 主色调 */
    --secondary-color: #764ba2;    /* 次色调 */
    --accent-color: #f093fb;       /* 强调色 */
    --bg-dark: #0f0f23;            /* 深色背景 */
    --bg-light: #1a1a2e;           /* 浅色背景 */
}
```
